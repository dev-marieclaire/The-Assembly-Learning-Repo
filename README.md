# The Assembly Language Learning Repo
This repository is dedicated to documenting my process of learning Assembly Language for multiple architectures and assemblers.
This may attach resources for the people interested on beginning into learning low-level stuff. (Webdevs, please, go away).

## Recommended reads and references.
### Introduction to Assembly.
  - Assembly Language for Intel-Based Computers by Kip R. Irvine, 4th edition `ISBN:0130910139`
  - Same book but in the 6th, 7th or 8th edition for more modern features and hardware.

### References for whenever you get to code (you must have previous knowledge).
  - Code examples for GNU Assembler: https://cs.lmu.edu/~ray/notes/gasexamples/
