# The Assembly Language Learning Repo
This repository documents my process of learning Assembly Language for multiple architectures and assemblers.

It may contain resources for people interested in getting started with low-level programming. (Web developers, please go away).

## Why would someone learn such an unusual language?
The purpose of this section is not to display pedantry or arrogance, but honestly, the only people who could possibly take this seriously or perceive skill-farming are vintage computing enthusiasts `(like me!)`, embedded systems enthusiasts, and computer science weirdos `(like me!)`.

You!, as a competent programmer, should be aware that every programming language has its limitations and weaknesses for many situations that could be required in a single project.

For example, you wouldn't use Python to create a sound driver. Similarly, you wouldn't use Assembly to create accounting software with a GUI.

You might accomplish these tasks through the use of dark magic or mysterious artefacts unknown to mankind, but you would have noticed that the level of difficulty would have been lower if you had simply switched languages for both projects.

Now, even though this may sound discouraging, I think it's worth taking a look at projects where Assembly language or low-level programming is a mandatory skill.
  - https://github.com/torvalds/linux
  - https://github.com/ffmpeg/ffmpeg
  - https://github.com/kraj/musl
